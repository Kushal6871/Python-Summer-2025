# Python-Summer-2025
Mini Python projects and exercises completed during Summer 2025 to practice data analysis, visualization, and basic programming skills.
## About
This repository contains a collection of small Python projects and practice exercises I completed to improve my programming abilities. It includes work with libraries such as pandas, numpy, and plotly for data manipulation and visualization.

It also contains my final project, which applies these skills to a comprehensive data analysis case study.
## What's Inside
- Data analysis and visualization mini-projects  
- Python scripting and problem-solving exercises  
- Exploration of datasets using pandas and plotly  
- Final project with detailed analysis and visualizations
